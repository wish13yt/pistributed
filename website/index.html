Pistributed

Pistrubuted is an open-source server & client to calculate Pi with distrubuted machines. Get started by downloading the client and connecting to a server.

No Pistributed servers are hosted by me except for the test server I use (which isn't public).

Make sure you trust the server you are connecting to! I am not responsible for what servers host.

Pistributed made with Flask, Python, and love ❤️

Client, server, and website is made available via GPL-3.0. Learn more at https://github.com/wish13yt/pistributed.
